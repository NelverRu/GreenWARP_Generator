## Возможности:  

- Генерация конфига в один клик  

- Выбор конечной точки:  
- [RU] Cloudflare WARP  
- [FR] Roubaix  
- [PL] Warsaw  
- [DE] Frankfurt  

- Автоматическая регистрация в Cloudflare WARP  


## Как использовать:  

1. Выберите конечную точку из списка  
2. Нажмите «Сгенерировать»  
3. Скачайте готовый `.conf` файл  
4. Импортируйте в AmneziaWG
